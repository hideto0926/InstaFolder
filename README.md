# instaFolder — landing page

Static site for the **instaFolder** iOS app (on-device photo organizer that suggests folders from image content + metadata, recognizes Instagram handles, and helps you free up space).

- `index.html` — landing page (EN / 日本語 toggle)
- `privacy.html` — privacy policy (EN / 日本語)
- `assets/` — app icon and screenshots
- `.nojekyll` — serve `assets/` as-is on GitHub Pages

## Publish with GitHub Pages
1. Push this `web/` folder to a repo (as the repo root, or `/docs`).
2. Repo ▸ Settings ▸ Pages ▸ deploy from branch.
3. Set the folder to the one containing `index.html`.
